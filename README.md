## ✨ Features
- **Live Internet Radio** – Enter any streaming URL (MP3, AAC, etc.) or upload an audio file.
- **Voice & Video Recording** – Record directly from your browser (microphone/camera).
- **AI Analysis (Mock)** – Generates a report with simulated speech clarity, duration, file size, and sentiment.
- **Download Reports** – Save analysis as `.txt` file.

## 🚀 Deployment on Streamlit Cloud
1. Push this repository to GitHub.
2. Log in to [Streamlit Cloud](https://streamlit.io/cloud).
3. Click **New app** → select your repo → set `app.py` as main file.
4. Add `haiti_flag.png` to the same folder (optional but recommended).
5. Deploy – your app will be live.
